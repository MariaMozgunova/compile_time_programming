## About The Project

This project evaluates polynomial at compile-time with constexpr. The program is also tested with Google Test.

<!-- GETTING STARTED -->
## Getting Started

To run the program on your machine, follow the simple steps below

### Installation

1. Clone the repo `git clone https://github.com/MariaMozgunova/compile_time_programming.git`
2. Include `evaluate_polynomial.hpp` into your code
3. Instantiate the `Polynomial` class
4. Call the `evaluate` function

You can find an example of using `evaluate_polynomial.hpp` at the `tests.cpp` file

To run tests with Google Test, you need to set up the Google Test library first. For example, you can find out how to do it with vscode at [this answer on Stack Overflow](https://stackoverflow.com/a/62911654/13907973)
